## [1.0.0] - 2024-01-01

### Added
- ESP32-CAM firmware with Wi-Fi AP, BLE, and MJPEG camera streaming
- HC-SR04 ultrasonic proximity sensor with dynamic buzzer alerts
- Raspberry Pi client with OpenCV, pyttsx3, and Gemini API integration
- React Native companion app (Expo SDK 50) with five main screens:
  - Dashboard with BLE connection status and telemetry
  - Camera OCR screen with text and object detection modes
  - Voice-guided navigation with preset destinations
  - AI voice assistant with chat interface and media player
  - Settings for API key, speech rate, and sensor calibration
- Interactive web simulator with hardware viewport and phone emulator
- Google Gemini 1.5 Flash integration for OCR, scene description, and chat
- BLE service for device scanning, connection, and telemetry streaming
- TTS voice service with configurable speech rate
- Comprehensive README with architecture diagrams and setup guides
- Wiring diagrams for ESP32-CAM and Raspberry Pi builds
- MIT license
